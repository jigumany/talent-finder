# **App Name**: Staffable

## Core Features:

- Secure Authentication: User authentication and authorization, with role-based access control for clients and candidates.
- Candidate Marketplace: Candidate profiles display, searchable/filterable by role, subject, qualifications, location, and availability.
- Availability Calendar: Availability calendar for clients to select dates and request candidate bookings, and for candidates to block out unavailable days.
- Booking Management: A streamlined interface for clients to view and manage past bookings, with options for rebooking previous candidates.
- Timesheet: Simple timesheet management that will record the time for billing.  The goal of the app is only to create the entries. No attempt to process those records or integrate billing is included in this MVP
- Review Generator: AI powered "tool" to help user write targeted and personalized reviews based on previous performance.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust, reliability, and professionalism, fitting for the education sector.
- Background color: Very light blue (#E8EAF6), close in hue to the primary color, but heavily desaturated to provide a clean backdrop.
- Accent color: Vibrant purple (#9C27B0), an analogous hue to the primary color, used to highlight key interactive elements and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern and accessible look that ensures readability on various screen sizes.
- Simple and clear icons that represent different roles, subjects, and functions within the app, enhancing usability.
- Clean and intuitive layout with clear sections for candidate listings, booking management, and profile settings. Use of whitespace to avoid clutter.
- Subtle animations for transitions and loading states to enhance user experience without being distracting.
